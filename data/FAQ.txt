8. HỌC VIÊN CẦN TỰ MANG LAPTOP HAY SỬ DỤNG MÁY TÍNH CỦA NHÀ TRƯỜNG?
================================================================================

TRƯỜNG TRẢ LỜI:

TỰ CHUẨN BỊ. Học viên cần tự chuẩn bị máy tính (laptop) để phục vụ việc học tập và thực tập.

LÝ DO:
- Máy tính là công cụ cá nhân
- Cần quen thuộc với máy của mình
- Sẽ sử dụng cả trong và ngoài giờ học

CẤU HÌNH TỐI THIỂU ĐỀ NGHỊ:

1. CPU (Bộ xử lý)
   - Intel Core i7 (thế hệ 8 trở lên)
   - HOẶC Apple M2 hoặc tương đương
   - HOẶC CPU cao cấp hơn

2. RAM (Bộ nhớ)
   - Tối thiểu: 16GB
   - Lý do: Khi học AI thường mở cùng lúc:
     * VS Code / PyCharm (IDE lập trình)
     * Jupyter Notebook (viết code)
     * Browser với nhiều tab
     * Tài liệu
     * Zoom (nếu học online)
     * Môi trường Python hoặc Docker
   - Nếu dưới 16GB sẽ chậm và lag

3. Ổ cứng SSD
   - Tối thiểu: 256GB
   - Vì sao:
     * Dataset (dữ liệu) có thể chiếm 5-10GB
     * Python packages (thư viện) chiếm 2-5GB
     * Model files (mô hình đã train) chiếm 1-5GB
     * Docker images (ảnh ảo) chiếm 5-10GB
     * OS + chương trình khác: 50GB

4. Hệ điều hành
   - Windows 10/11
   - HOẶC macOS
   - HOẶC Linux

5. Kết nối mạng
   - Ổn định để học online
   - Đủ tốc độ để đồng bộ dữ liệu
   - Hỗ trợ sử dụng các nền tảng AI/cloud

LƯU Ý:
- Chương trình không bắt buộc máy quá mạnh
- Vì nhiều nội dung có thể chạy trên môi trường cloud
- Hoặc tài nguyên do chương trình hỗ trợ
- Nhưng cấu hình trên là tối thiểu để học ổn định

3. PHÁT TRIỂN TRONG MÔI TRƯỜNG DOANH NGHIỆP
   - Đối với học viên có thành tích XUẤ SẮC

- Năng lực thể hiện trong quá trình tham gia dự án
